Questão 1

Analise a tabela a seguir e realize e obtenha os respectivos resultados


1.1 Conforme a tabela a cima qual é a Query necessária para exibir apenas a linha do procedimento 'TC ABDOME TOTAL'
Pode-se utilizar a seguinte query para exibir apenas a linha do procedimento 'TC ABDOME TOTAL':

SELECT *
FROM nome_tabela
WHERE ProcedureName = 'TC ABDOME TOTAL';

1.2 Para realizar a exclusão de todas as linhas é utilizado a função DELETE no banco de dados, demonstre a estrutura deste comando para realizar a exclusão de todas as linhas exceto o número de acesso 499270
Para deletar conforme o enunciado, pode-se utilizar a query a seguir:

DELETE FROM nome_tabela
WHERE AccessionNumber != 499270;


Questão 2

Analise a tabela a seguir e realize e obtenha os respectivos resultados

2.1 Observando que são 2 tabelas diferentes, 'Requests' e 'study' Qual seria a query necessária para ser exibido a linha do número de acesso 499305 das duas tabelas na mesma linha.
Pode-se utilizar a seguinte query:

SELECT *
FROM Requests
JOIN study
ON requests.AccessionNumber = study.accession_no;
